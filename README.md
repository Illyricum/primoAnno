# primoAnno

Raccolta di appunti personali presi durante l'anno accademico, organizzati per corso.

## Contenuto

Questo repository contiene appunti, sbobine e note personali relativi ai corsi
del primo anno del corso di laurea in Ingegneria Informatica presso l'Universita di Pisa.

## Struttura

Gli appunti sono divisi per corso, ciascuno in una cartella dedicata:

primoAnno/
- Analisi-2/
- Basi-di-Dati/
- ...altri corsi

## Avvertenze

- Questi appunti sono materiale personale di studio e potrebbero contenere
  imprecisioni,errori concettuali o omissioni.: non sostituiscono i libri di testo o le lezioni ufficiali.
- Parte del contenuto puo derivare o rielaborare materiale fornito dai docenti
  (slide, dispense), che rimane di proprieta dei rispettivi autori/istituzioni.
